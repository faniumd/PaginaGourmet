# Fusion Gourmet - Avant-Garde Gastronomic Festival

## Description
A modern and elegant landing page for a high-end culinary festival. This project focuses on semantic HTML5 structure, sophisticated CSS3 styling, and fully responsive design.

## Technologies Used
* HTML5 (Semantic tags)
* CSS3 (Flexbox, Grid, Media Queries)
* Google Fonts (Playfair Display & Poppins)

## Project Structure
* `index.html`: Main structure and content.
* `style.css`: Advanced styling and layout.
* `assets/`: Images and multimedia (simulated).

## Instructions
1. Download the files.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
